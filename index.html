Faq

What is Instep?

Instep is a next-generation Roblox anti-exploit solution that is designed to block all types of injections, code executions, and exploits.

Does Instep have any impact on game performance?

Instep has minimal impact on game performance and is designed to run in the background without interfering with gameplay. There will be no FPS impact on client side and almost 0 ms delay at server side.

Where can I test the Instep?

Can I use Instep for all of my games?

Yes, the key is assigned to your UserId. Once you buy the anti-cheat, you will be able to use it on each one of your Roblox games.
